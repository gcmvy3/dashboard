    Detailed Description

    The StereoWidget class is a custom widget which displays stereoscopic visual data obtained from shared memory.

    This widget contains a StereoMemory child widget, which displays the QImage frames generated by StereoMemory using data from shared memory. This widget creates and manages its child widgets automatically upon construction of the widget.

    See also StereoThread and StereoMemory.
