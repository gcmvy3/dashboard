    Detailed Description

    The StereoThread class is a custom class which manages packets for the Stereo Widget.

    A single StereoThread object is used to read all packets containing indicators of new data from the Voltron Core process. The data from these packets are extracted and sent to the corresponding StereoMemory object(s).

    See also StereoMemory and StereoWidget.

    Member Function Documentation

    void StereoThread::readPendingDatagrams()

    Executed when the UDP socket associated with this class signals that there is data to be read from the Voltron Core process.

    Upon execution, the available data is read into a QByteArray from the UDP socket and passed to the processDatagram(QByteArray datagram) function.
