    Detailed Description

    The BatteryThread class is a custom class which manages packets for the Battery Widget.

    A single BatteryThread object is used to read all packets containing battery data from the Voltron Core process. The data from these packets are extracted and sent to the corresponding BatteryWidget object(s).

    Note: While this and the BatteryWidget classes are fully implemented, the Voltron Core process, at present, does not send true battery data.

    See also BatteryWidget.

    void BatteryThread::readPendingDatagrams()

    Executed when the UDP socket associated with this class signals that there is data to be read from the Voltron Core process.

    Upon execution, the available data is read into a QByteArray from the UDP socket and passed to the processDatagram(QByteArray datagram) function.
