    Detailed Description

    The LidarWidget class is a custom widget which displays 3-d point cloud visual data obtained from shared memory.

    This widget contains a LidarRenderer child widget, which displays view of the 3-d space generated using data from shared memory. This widget creates and manages its child widgets automatically upon construction of the widget.

    See also LidarThread and LidarRenderer.
