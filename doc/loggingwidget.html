    Detailed Description

    The LoggingWidget class is a custom widget which allows the user to control how data is recorded by the Voltron Logging process.

    This widget contains multiple QPushButton child widgets, which when pressed will result in the LoggingThread sending a packet to the Logging process, telling it to start or stop recording the corresponding data from the Core process.

    See also LoggingThread and LoggingManager.
