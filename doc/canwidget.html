    Detailed Description

    The CANWidget class is a custom widget which displays the value of certain CAN codes in real time.

    This widget shows a list of all CAN codes currently loaded by CANCodeManager. It is updated via signals when new CAN codes are loaded. When CANThread receives an updated CAN value, it emits a signal that updates every CANWidget automatically.

    See also CANThread, CANCode, and CANCodeManager.
