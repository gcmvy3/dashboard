    Detailed Description

    The BatteryWidget class is a custom widget which displays data sent to it in Battery packets.

    This widget contains a QTableWidget child widget, which displays the battery charge data sent to the BatteryWidget via signals. The data is displayed in a table with each row corresponding to a charge value. Battery packets sent to this widget specify which row of the table will be updated with the provided charge value.

    Note: While this and the BatteryThread classes are fully implemented, the Voltron Core process, at present, does not send true battery data.

    See also BatteryThread.

    [protected] void BatteryWidget::onPacket(BatteryPacket packet)

    Executed when a BatteryThread object signals a new Battery packet to be processed.

    Updates the cell of the child QTableWidget table indicated by the packet struct with the charge value also provided by the packet struct.
