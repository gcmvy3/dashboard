    Detailed Description

    The CommunicationManager class is a custom class which manages all the QThreads and thread objects being used to communicate with the Voltron Core process.

    This class instantiates the various thread objects handling the different packets being sent by the Voltron Core process and moves each to its own QThread. Connections are established to enable communication between the thread objects and the QThreads they have been moved to, and each QThread is started.

    See also QThread.
