    Detailed Description

    The ConsoleWidget class is a custom widget which displays messages sent to it in Console packets.

    This widget contains a QTextBrowser child widget, which displays the messages sent to the ConsoleWidget via signals. All messages are displayed with a time stamp of when the message was received by the ConsoleWidget.

    See also ConsoleThread.

    Member Function Documentation

    [protected] void ConsoleWidget::errorString(int error)

    Calls the message handler with the critical error error.

    See also qCritical().

    [protected] void ConsoleWidget::onPacket(DebugPacket packet)

    Executed when a ConsoleThread object signals a new Console packet to be processed.

    Generates a QString composed of a time stamp followed by the message stored in the packet struct. This QString is then displayed in the child QTextBrowser widget of the Console widget.

    [protected] void ConsoleWidget::onStopReading()

    Disconnects the widget from the incoming data packets for better performance.

    [protected] void ConsoleWidget::showEvent(int *event)

    This function overloads showEvent( QShowEvent* event ).

    This function is automatically called when the widget is shown. Connects the widget to the incoming data packets.

    See also QWidget::showEvent( QShowEvent* event ).
