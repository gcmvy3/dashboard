    Detailed Description

    The LoggingThread class is a custom class which manages packets being sent to the Voltron Logging process.

    A single LoggingThread object is used to send command packets to the Voltron Logging process. These packets tell the Logging process to start or stop logging certain sets of data from the Core process.

    See also LoggingWidget and LoggingManager.

    Member Function Documentation
