    Detailed Description

    The LoggingManager class is a custom class which determines which deteremines what code to send to the Logging process via LoggingThread based on input from LoggingWidget.

    This class ensures that the Logging process records only what the user tells it to through input on the dashboard. When a button is pressed in the LoggingWidget widget, the LoggingManager will create a packet containing a code corresponding to that button press. This packet is then sent by the LoggingThread object to the Logging process to start or stop logging of the corresponding data set.

    See also LoggingThread and LoggingWidget.
