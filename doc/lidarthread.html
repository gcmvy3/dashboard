    Detailed Description

    The LidarThread class is a custom class which manages packets for the Lidar Widget.

    A single LidarThread object is used to read all packets containing indicators of new data from the Voltron Core process. The data from these packets are extracted and sent to the corresponding LidarRenderer object(s).

    See also LidarRenderer and LidarWidget.

    Member Function Documentation

    void LidarThread::readPendingDatagrams()

    Executed when the UDP socket associated with this class signals that there is data to be read from the Voltron Core process.

    Upon execution, the available data is read into a QByteArray from the UDP socket and passed to the processDatagram(QByteArray datagram) function.
